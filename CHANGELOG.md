# Changelog

All notable changes to this project will be documented in this file.

## [4.0.0] - 2026-06-10

### Features

- Streakr v4 redesign with sidebar + mobile bottom navigation
- Today card grid with drag-to-reorder (SortableJS)
- Calendar month view with per-habit colored dots
- Stats view with heatmap and import/export JSON
- Confetti celebration when all habits complete
- PWA manifest and service worker for offline use

### Compliance

- Spec-Kit (`.specify/` and `.speckit/`) scaffolding
- GitLab CI with secret scan and Pages deploy
- AGPL-3.0 license and project documentation
