# Contributing to Streakr

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository.
2. Clone your fork locally.
3. Create a branch: `git checkout -b feat/my-feature`
4. Make changes following the guidelines below.
5. Commit using [Conventional Commits](https://www.conventionalcommits.org/): `git commit -m "feat: add feature"`
6. Push and open a Pull/Merge Request.

## Development

Open `index.html` in a browser or use any static file server. No install step required.

## Code Style

- **JavaScript:** ES6+, vanilla only.
- **CSS:** Follow existing design tokens in `style.css`.

## Pre-commit Hooks (optional)

```bash
pip install pre-commit
pre-commit install
pre-commit run --all-files
```

## Commit Messages

- `feat:` — New feature
- `fix:` — Bug fix
- `docs:` — Documentation
- `chore:` — Tooling / CI

## Reporting Issues

Include steps to reproduce, expected vs actual behavior, and browser/OS details.
