# Streakr — User Manual

## Getting Started

### What is Streakr?

Streakr helps you build consistent habits. Add habits with emoji and colors, check them off daily, and watch your streaks grow on the calendar and stats views.

### How to Use

1. **Open Streakr** in your browser (or install as a PWA from the browser menu).
2. **Add a habit** — click the **+** button (FAB on mobile).
3. Pick an **emoji**, **color**, **name**, and **days per week** target.
4. On the **Today** tab, tap the circle to mark a habit complete.
5. Switch to **Calendar** to toggle any past day.
6. Open **Stats** for streak summaries, heatmaps, and **Export/Import** backup.

### Views

| View | Description |
|------|-------------|
| Today | Card grid, streak pill, 7-day pips, completion ring |
| Calendar | Month grid with colored dots per habit |
| Stats | Summary tiles, per-habit %, 84-day heatmap |

### Backup & Restore

- **Export** downloads `streakr_backup_DATE.json`.
- **Import** merges or replaces habits from a JSON file.

### PWA / Offline

Install from your browser's "Add to Home Screen" option. The service worker caches static files for offline use. Data remains in local storage on your device.
